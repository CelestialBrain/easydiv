EasyDiv settings

Configure custom colors and extraction behavior.

Tailwind version

Controls which token names are emitted. v4 renames several defaults (shadow → shadow-sm, bg-gradient-to-r → bg-linear-to-r, etc.). Most class names are identical in both; this only affects the ones that diverged.

Custom color palette

Paste your tailwind.config.js colors so captures map to bg-primary / text-brand-500 instead of arbitrary rgb values. Accepts JSON, or the colors: block from a Tailwind config.

Example: pasted directly from tailwind.config.js 
colors: {
  primary: '#3b82f6',
  brand: {
    DEFAULT: '#3b82f6',
    light: '#bfdbfe',
    dark: '#1e40af'
  }
}

Detected colors

No custom palette loaded — Tailwind v3 defaults will be used.
